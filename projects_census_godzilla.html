Census Godzilla


Census Godzilla lets the user visualize any U.S. Census American Community Survey dataset and provides contextual socioeconomic indicators. This feature allows the user to conduct preliminary exploratory analysis on how data is distributed across space in relation to their main census variable that they have selected.
